# guestbook
A simple questbook  as  XML file database

Гостевая книга (Ajax и XML)
В данный момент  это только  модель, демонстрирующая основные  возможности.

БД -  XML файл.
На сервере записываются сообщения и ответы. 
Присутствует возможность установить время задержки  м/у последовательным отправлением сообщений,
времнем обновления документа( используется, чтобы пользователь видел другие ответы пользователей).
Необходимо установить дополнительные проверки.
